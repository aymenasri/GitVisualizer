# 🚀 GitVisualizer

Un visualiseur de profil GitHub magnifique et interactif construit avec des technologies web modernes. Cette application vous permet de rechercher n'importe quel utilisateur GitHub et de voir instantanément ses statistiques, ses dépôts (repos) et une belle visualisation de ses langages de programmation les plus utilisés.

## ✨ Fonctionnalités

- **Recherche de Profil** : Cherchez instantanément n'importe quel pseudo GitHub.
- **Carte Développeur** : Visualisez les informations clés (followers, following, repos totaux, bio, localisation).
- **Analyse des Langages** : Un graphique circulaire (Pie Chart) dynamique détaillant les langages de programmation favoris de l'utilisateur à travers ses dépôts publics.
- **Interface Moderne** : Un design épuré inspiré du "Glassmorphism" avec un thème sombre élégant.

## 🛠️ Technologies Utilisées

- **React.js** (v19)
- **Vite** (Environnement de développement ultra-rapide)
- **Tailwind CSS v4** (Framework CSS utilitaire)
- **Recharts** (Librairie graphique construite sur React)
- **Axios** (Pour les requêtes HTTP vers l'API GitHub)
- **Lucide React** (Icônes magnifiques)

## 📦 Démarrage Rapide

### Prérequis

Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre ordinateur.

### Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/aymenasri/GitVisualizer.git
   ```
2. Aller dans le dossier du projet :
   ```bash
   cd GitVisualizer
   ```
3. Installer les dépendances :
   ```bash
   npm install
   ```
4. Lancer le serveur local :
   ```bash
   npm run dev
   ```

Vous pouvez maintenant utiliser l'application depuis votre navigateur sur `http://localhost:5173`.
